¡Hola, bienvenido! Este es un proyecto personal donde exploro historias ocultas y mitos enigmáticos del mundo de los videojuegos.

*Sobre el proyecto*
El objetivo de esta página es ofrecer un sitio interactivo y visualmente atractivo para los entusiastas del gaming. En ella podrás encontrar:
-Historias ocultas: Detalles sobre mitos urbanos populares.
-Interfaz responsiva: Diseñada para verse bien en computadoras y dispositivos móviles.
-Efectos interactivos: Uso de JavaScript para mejorar la experiencia del usuario.

Tecnologías utilizadas
Para el desarrollo use:
-HTML5.
-CSS3.
-Javascript.
